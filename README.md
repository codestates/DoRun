# DoRun's Feature
* 지도 서비스를 통해 원하는 위치를 쉽게 확인할 수 있습니다
* 회원가입 없이도, 이미 생성돼 있는 크루 정보를 확인할 수 있습니다.
* 직접 크루를 모집할 수도 있으며, 기본 제공되는 템플릿을 통해 간편한 조건 설정이 가능합니다.(회원가입 필수)
* 같은 크루에 참여한 사람들(mate) 간에 실시간으로 소통할 수 있도록 채팅 기능을 지원합니다.
* 지정된 종료 시간 이후, 크루와 채팅방의 기록이 완전히 삭제 됩니다.
* 지정된 종료 시간 이전에도 My Page를 통해 자유로운 크루(및 채팅방) 탈퇴가 가능합니다.
